OptClicker

Option → Right Click.

Overview

Right-click with the Option key.

Press Option to simulate a right-click. Hold it for a right-click hold — perfect for aiming in games. Designed for gaming with a touchpad, OptClicker lets you left and right click simultaneously without a physical mouse.

Features

Simple, smart, automatic.

01 / Auto Toggle

Enable Per App

Automatically enable OptClicker when specific apps, websites, or spaces are active. Match by bundle ID, process name, or browser URL.

02 / Shortcuts

Global Hotkeys

Toggle OptClicker or Auto Toggle with customizable global shortcuts. Live key recording — press any combination to assign.

Menu Bar App

Runs quietly in the menu bar. Dynamic icon shows enabled/disabled state, with an "A" indicator when Auto Toggle is active.

Auto Toggle Rules

Match by bundle ID, exact or fuzzy process name, or browser URL. Combine with space-based detection for per-workspace control.

Install

Get started in seconds.

macOS 13.0+. Drag to Applications, done.

Download
or brew install --cask gitmichaelqiu/tap/optclicker

Not signed with a paid Apple Developer account — go to System Settings → Privacy & Security if macOS blocks the app. Permissions are guided on first launch.
