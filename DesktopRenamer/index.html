Overview

Take back control of your macOS spaces.

DesktopRenamer is a macOS menubar app that lets you give custom names to your desktop spaces — those virtual desktops you create in Mission Control.

Quickly rename from the menu bar. See large, aesthetic labels when you enter Mission Control. Switch spaces with zero-delay overrides. Automate everything via AppleScript or the built-in HTTP API.

Features

Everything you need to own your workflow.

Rename at Menu Bar

Quickly give your desktop spaces a custom name directly from the menu bar. Click the menubar icon, type a name, and it's set — no complicated setup, no configuration files.

Crystal Clear Labels

See large, aesthetic name labels when you enter Mission Control, so you instantly know which space is which. When you switch spaces, a discreet active label appears briefly to confirm where you landed. Both can be toggled independently to match your preference.

Switch & Move

Click any space in the menu bar to jump directly to it. Hold the Option (⌥) key while clicking to instantly teleport the active window to that space. No more dragging windows across desktops.

Faster Switching

Bypass native macOS animation lag. Enable trackpad gesture overrides or configurable keyboard hotkeys for instant, zero-delay switching between spaces. Your muscle memory, supercharged.

Raycast Integration

Are you a power user? DesktopRenamer integrates directly with Raycast, letting you manage and switch spaces elegantly via your favorite launcher — without ever touching the menu bar.

API & AppleScript

DesktopRenamer exposes a full AppleScript dictionary for automating space management — rename spaces, switch desktops, move windows, all from your scripts. The built-in SpaceAPI lets other apps acquire space IDs and names over HTTP. Toggle it on in Settings → Advanced.

Desktop Widget

Add the DesktopRenamer widget to your Notification Center or Desktop. See your current space name at a glance, browse all configured spaces, and tap to switch — directly from the widget. Available in small, medium, and large sizes with both standard and transparent backgrounds.
Install

Get started in seconds.

Direct Download

Download the latest version from GitHub Releases, drag to Applications, and you're done. No SIP disabling required. Requires macOS 13.0 Ventura or later.

Download from GitHub
Homebrew

Prefer the command line? Install via Homebrew with a single command.

brew install --cask gitmichaelqiu/tap/desktoprenamer
First Launch

Since DesktopRenamer isn't signed with a paid Apple Developer account, macOS may show a "Developer not verified" alert. Go to System Settings → Privacy & Security → Open DesktopRenamer to allow it. If issues persist, run:

codesign --force -s - /Applications/DesktopRenamer.app
Permissions

DesktopRenamer needs Accessibility permission for hotkeys and trackpad overrides, and Automation permission to control System Events for Mission Control. You'll be guided through granting these on first launch.
